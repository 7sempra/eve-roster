# eve-roster back-end

## Get started

1. Install the latest version of [Node.js](https://nodejs.org/en/).
2. `$ cd <this dir>`
3. `$ npm install`
4. Create a file named `config.local.json`. Fill in any config values that are `null` in `config.json`.

Then:

``` bash
# create empty database
$ npm run create-db
```

``` bash
# serve at localhost:8081
$ npm start
```
